Desearch use case prototype

eBay price intelligence from web search + crawl

This demo intentionally uses the SERP Web Search endpoint, not AI Search, because GET /web preserves operators like site:ebay.com. Then it crawls selected listings and extracts price signals.

Try the flow OpenAPI 
GET /web?query=site%3Aebay.com%20iPhone%2015%20Pro%20Max&start=0

GET /web/crawl?url={listing}&format=text
